- **技术栈**：采用 Vue 3 (Composition API) + Vite 构建，使用 Pinia 进行状态管理，Vue Router 处理路由，Element Plus 作为 UI 组件库。
- **目录结构**：遵循标准 Vue 项目结构，`src/api` 封装 Axios 请求，`src/stores` 管理全局状态（认证、通知），`src/views` 按业务模块（用户、管理员、活动）组织页面，`src/layouts` 定义主布局和管理员布局。
- **网络层**：通过 `src/api/http.js` 创建 Axios 实例，统一处理请求拦截（自动携带 Token）和响应拦截（统一错误提示、401 跳转登录）。
- **路由守卫**：在 `src/router/index.js` 中实现基于角色的访问控制（RBAC），通过 `meta` 字段区分游客、登录用户和管理员权限。
- **实时通信**：在 `MainLayout.vue` 中集成 WebSocket 连接，用于接收实时通知并更新未读计数。
